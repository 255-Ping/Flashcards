This flashcard application is made for a school I worked at from November 2024 to present.
It was developed to help students review math-related material efficiently.

Round Menu:
  ➡ Start: starts the round
  ➡ Settings: Update card generation and application settings
  ➡ Statistics: View student data on a linegraph
  ➡ Decks: View deck editor
  ➡ Seq Edit: Under construction

Settings:
  ➡ Student: Select a student for data to be recorded
  ➡ Operator: Select an operator for random card generation
  ➡ Req. #: Select a number for random card generation
  ➡ Max Questions: Select the maximum questions for random card generation
  ➡ Card Generation: Select the card generation to be used

Statistics:
  ➡ Student: Select a student to view data for
  ➡ Graph Visuals: Check boxes to update how the graph looks
  ➡ Graph Filters: Check boxes to filter for specific data

Deck Editor(Decks):
  ➡ Deck: The name to be used for a new deck, cannot contain spaces
  ➡ Cards: The cards to be initially added to the deck
    ➡ Card Text Format: 1+1,2-2,3*3,4/4
    ➡ Cannot contain spaces
    ➡ Each card can only have 2 values
    ➡ Each card can only have 1 operator
    ➡ Must be seperated by a comma
  ➡ Deck UI: The visualization of a deck
    ➡ Deck Name: View or Change the name of the deck
    ➡ Select: Click to select a deck to be used
    ➡ Add: Click to add card from "Add Text Box" to deck
    ➡ Add Text Box: Card(in text form) to be added to deck
    ➡ Delete: Delete the deck
    ➡ X: Delete the card

Keybinds:
  ➡ ENTER: Submit answer and Submit settings
  ➡ CTRL + QUOTELEFT: Exit application
  ➡ CTRL + Q: Unlock mouse cursor after end of round
  ➡ CTRL + N: Generate new flashcard(DEBUG)
  ➡ CTRL + E: Forcibly end round(DEBUG)
  ➡ CTRL + P: Create a test popup(DEBUG)
